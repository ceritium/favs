# favs

Store you favourite commands

## Installation

```
git clone git@github.com:ceritium/favs.git
cd favs
make install
```

## Usage

```
favs --help

Usage:
  favs [flags...] [arg...]

Store your fav commands

Flags:
  --help     # Displays help for the current command.
  --version  # Displays the version of the current application.

Subcommands:
  add, a
  list, ls
  show
```
